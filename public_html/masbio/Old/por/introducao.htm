Introdução

Não existe projeto de seqüenciamento de genoma que escape à velocidade imposta pela necessidade de finalizar a anotação e o seqüenciamento do organismo alvo no menor tempo possível. Acreditamos que exista uma carência de ferramentas capazes de automatizar atividades mecânicas, não permitindo assim que os especialistas tenham tempo para executar tarefas que realmente requeiram seu conhecimento. No entanto, existem poucas ferramentas que automatizam as etapas envolvidas no processo de seqüenciamento e anotação de genoma. Em particular, faltam ferramentas integradas.

O objetivo principal deste projeto é a implementação de um ambiente para anotação automática de genomas. Este ambiente é denominado ATUCG - Ambiente para anoTação aUtomátiCa de Genomas. O ATUCG está integrado a outros sistemas que visam o desenvolvimento de ferramentas para análise de dados com o uso de tecnologia de agentes.
