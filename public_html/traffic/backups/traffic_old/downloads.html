Simulator

If you want to download our traffic simulator, please send us an email informing you name, institution and interests. Although everything we develop is free software, we would like to keep in touch with everyone who's using our software.
In order to receive a copy of beta (unstable) version of the simulator, please contact the project coordinator at MASLAB. Please notice that currently the simulator runs under Linux only. Besides, this is a volunteer work; we cannot provide much help with installing and running the simulator. Also, it is necessary to have the following software installed:

gcc
python
qt3
qt3-devel
PyQT (the Python bindings for creating QT3 applications)
In some Linux distros it is called kdebindings3-python or python-qt3
kdebase3-devel (on Suse)
python-xml
